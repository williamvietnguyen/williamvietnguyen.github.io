# My Personal Site

### Requirements:

- Ruby version 2.4.0 or higher
- RubyGems
- GCC and Make
- Jekyll
- Bundler

### Set up:

Run the following to run locally on localhost:4000.

1. `bundle install`
2. `bundle exec jekyll serve`
